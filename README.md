**ระบบจัดการสินค้าออนไลน์ (Online Inventory System)**
พัฒนาโดย: **นายเอกอมร [6540205801]**

ระบบจัดการสินค้าออนไลน์นี้พัฒนาขึ้นเพื่อใช้ในการบริหารจัดการสินค้าภายในร้านค้าออนไลน์ โดยมีความสามารถในการเพิ่ม แก้ไข ลบ และค้นหาข้อมูลสินค้าได้อย่างมีประสิทธิภาพ นอกจากนี้ยังมีระบบสำหรับการขายสินค้าเพื่ออำนวยความสะดวกแก่ผู้ใช้งานในการดำเนินธุรกิจออนไลน์

เทคโนโลยีที่ใช้ในการพัฒนา:

* **Frontend:** พัฒนาโดยใช้ **Next.js (React Framework)**
* **Backend:** พัฒนาโดยใช้ **Bun และ Elysia.js**
* **Database:** ใช้ฐานข้อมูล **MongoDB**
